## Project structure

```
vimaan-dynamic/
├── index.html          # HTML shell — sections + script tags
├── css/
│   ├── main.css        # Design tokens, layout, all components
│   └── animations.css  # Keyframes and loader styles
├── js/
│   ├── audio.js        # AudioContext: ambient hum + beep()
│   ├── drone.js        # Three.js scene, drone mesh, render loop
│   ├── hud.js          # Four 2-D canvas HUD renderers
│   ├── incident.js     # Live incident canvas (wow section)
│   └── main.js         # Cursor, loader, GSAP scroll triggers
├── assets/
│   ├── fonts/          # (add local fonts here if needed)
│   └── icons/          # (SVG icons, favicons, etc.)
├── .gitignore
├── LICENSE
```
